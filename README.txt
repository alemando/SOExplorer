# SOExplorer
 Explorador de archivos SO
Para ejecutarlo debe realizar el siguiente paso a paso:
1. Abrir la carpeta SOExplorer por medio de la terminal de comandos

2. Estando en SOExplorer, ejecutamos el siguiente comando:
npm install, y esperamos que se descarguen todas las dependencias.

3. Posteriormente ejecutamos el siguiente comando, para iniciar la aplicación:
npm start

3. La consola nos mostrará un mensaje diciendo en qué puerto está el server
   habilitado (server on port...), por defecto está definido el puerto 3000
   (server on port 3000).

4. Para usarlo dentro del servidor solo basta con escribir en el explorador
   de archivos:
localhost:3000
   En caso de que el puerto por defecto cambie, solo hace falta cambiar el
   3000 por el puerto que indique la consola.

5. Para abrirlo por fuera del servidor solo hace falta escribir la dirección
   IP de la máquina dónde está alojada la carpeta SOExplorer, seguida del 
   número de puerto, por ejemplo:
192.168.1.65:3000
   
Luego de seguir estos pasos, el explorador de archivos estará funcionando
correctamente.
